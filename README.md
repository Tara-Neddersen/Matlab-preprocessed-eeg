# MATLAB-Python EEG Viewer

A clean, interactive EEG viewer that uses MATLAB for data processing and Python for visualization and seizure detection.

## Overview

This solution provides:
- **MATLAB processing**: Exact same pipeline as your original analysis
- **Python interface**: Interactive viewer with buttons and seizure detection
- **Clean separation**: No modifications to original code
- **Easy workflow**: Simple two-step process

## Quick Start

### Step 1: Process Data in MATLAB

1. Open MATLAB and navigate to this folder:
   ```matlab
   cd matlab_python_eeg_viewer
   ```

2. Run the data processor:
   ```matlab
   matlab_data_processor()
   ```

This will:
- Load your SW1245_P30SS data
- Apply the exact same processing as `openephysreaderjkk.m`
- Save processed data as `processed_data/eeg_data.csv`

### Step 2: Run Python Viewer

1. Run the interactive viewer:
   ```bash
   python eeg_viewer.py
   ```

This opens an interactive interface with:
- Navigation controls
- Seizure detection
- Manual marking
- Results saving

## Features

### Navigation Controls
- **Previous/Next**: Move through time windows
- **Jump to Time**: Enter specific time
- **Window Size Slider**: Adjust viewing window (5-120 seconds)

### Seizure Detection
- **Auto Detect**: Automatic seizure detection algorithm
- **Manual Mark**: Click to mark seizures manually
- **Clear All**: Remove all marked seizures
- **Save Results**: Export detection results

### Visualization
- **Dual Channel Display**: Channel 57 (blue) and Channel 61 (red)
- **Seizure Highlighting**: Red overlays for detected seizures
- **Auto-scaling**: Automatic y-axis scaling
- **Status Bar**: Current time, window size, seizure count

## MATLAB Processing Details

The `matlab_data_processor.m` script applies the exact same processing as your original analysis:

1. **Data Loading**: Uses `OpenEphysLoader` with 250 Hz resampling
2. **Unit Conversion**: Divides by 1000 to convert to mV
3. **Filtering**: 1-25 Hz bandpass filter (Butterworth, causal)
4. **Time Array**: Creates time array in seconds
5. **Data Export**: Saves as CSV for Python compatibility

## Python Viewer Details

The `eeg_viewer.py` script provides:

### Seizure Detection Algorithm
- **Peak Detection**: Finds peaks above threshold (2.0 mV)
- **Grouping**: Groups peaks within 5-second gaps
- **Duration Filter**: Minimum 2-second duration
- **Peak Count**: Minimum 5 peaks per seizure

### Manual Marking
- **Click Mode**: Click to mark seizure start and end
- **Visual Feedback**: Real-time status updates
- **Validation**: Prevents invalid (end < start) seizures

### Results Export
- **JSON Format**: Structured seizure data
- **Summary Report**: Human-readable summary
- **Timestamped**: Unique filenames with timestamps

## Troubleshooting

### MATLAB Issues
- **"Data directory not found"**: Check that the SW1245_P30SS data exists
- **"OpenEphysLoader not found"**: Ensure MATLAB paths are set correctly

### Python Issues
- **"Data file not found"**: Run MATLAB script first to generate data
- **Import errors**: Install required packages: `numpy`, `pandas`, `matplotlib`, `scipy`

### Viewer Issues
- **Slow performance**: Reduce window size or data length
- **Memory issues**: Process data in smaller chunks
